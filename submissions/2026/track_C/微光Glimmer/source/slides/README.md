# Glimmer hackathon slides

这是一份约 2 分钟的 Marp deck，重点讲技术路线和工程亮点，而不是普通产品功能介绍。

## 使用

```bash
cd slides
npm install
npm run html
npm run pdf
npm run pptx
```

输出文件会生成在 `slides/dist/` 下。

## 讲述节奏

- 第 1 页：总览，15 秒
- 第 2 页：训练策略，20 秒
- 第 3 页：混合精度 GGUF，25 秒
- 第 4 页：parity-first 工程，25 秒
- 第 5 页：指标，20 秒
- 第 6 页：analysis-to-explanation 与总结，30 秒
