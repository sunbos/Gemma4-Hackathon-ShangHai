# Label-free Proteomics Toy Data

This directory contains small synthetic label-free proteomics files for real limma container execution in the review demo.

- `metadata/sample_metadata.csv`: two control and two treatment samples.
- `quant/protein_intensity.csv`: protein-level intensity matrix.
- `database/proteins.fasta`: tiny protein FASTA reference.
- `spectra/*.mzML`: minimal mzML-like files for file-shape demonstration.

The data are synthetic and intentionally small. They exercise the analysis path but are not intended for statistical claims.
