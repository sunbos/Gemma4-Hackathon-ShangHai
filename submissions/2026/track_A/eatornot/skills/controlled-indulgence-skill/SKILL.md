# 控制放纵技能

当用户表达强烈食欲、情绪化进食或想犒劳自己时使用此技能。

## 规则

- 永远不要因为食欲而羞辱用户
- 在额度内允许控制性放纵
- 检测情绪化进食模式（疲惫、压力 = 高渴望）
- 如果放纵额度用完，建议替代方案
- 将放纵与后续较轻的餐食平衡

## 情绪检测

| 情绪 | 渴望水平 | 建议 |
|------|----------|------|
| 疲惫 | 高 | 允许适度放纵 |
| 压力大 | 高 | 允许适度放纵 |
| 开心 | 低 | 引导健康选择 |
| 正常 | 中 | 平衡建议 |

## 放纵管理

- 默认每周 2 次放纵额度
- 放纵 = 高热量、高脂肪、含糖项
- 跨周追踪

## 语言指南

- 使用："偶尔犒劳一下没问题"、"你值得享受"
- 避免："你不应该"、"这不健康"、"你会后悔的"
