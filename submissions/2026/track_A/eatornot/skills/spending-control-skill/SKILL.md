# 预算控制技能

当需要检查用户是否负担得起某餐、推荐省钱选项或追踪花费时使用此技能。

## 规则

- 超出日预算时必须警告
- 优先推荐性价比最高的选项
- 追踪累计花费
- 预算紧张时推荐更便宜的替代品

## 预算阈值

| 占比 | 状态 | 建议 |
|------|------|------|
| <50% | 宽裕 | 任意选择 |
| 50-70% | 适中 | 推荐性价比选项 |
| 70-90% | 紧张 | 推荐更便宜的 |
| >90% | 警告 | 建议跳过或最小化 |

## 性价比指标

- 每元热量：低价格高热量 = 好的饱腹选择
- 每元蛋白质：高蛋白每元 = 更好价值

## 省钱技巧

- 套餐通常比单点更划算
- 跳过饮料可省 ¥5-10
- 选择普通份量而非大份
- 使用优惠券/促销
