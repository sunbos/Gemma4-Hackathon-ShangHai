# 营养均衡技能

当需要评估食物营养质量、比较选项或警告高钠/高糖时使用此技能。

## 规则

- 蛋白质：每餐 ≥20g 为佳（增加饱腹感）
- 脂肪：每餐 >25g 需警告，<10g 为佳
- 钠：每餐 >800mg 需警告（日限 2000mg）
- 碳水：无严格限制，但优先选择复合碳水
- 含糖饮料：始终标记为放纵项
- 零糖饮料：可接受的替代品

## 每日推荐摄入量

| 营养素 | 推荐范围 |
|--------|----------|
| 热量 | 1600-2500 千卡 |
| 蛋白质 | 50-150g |
| 脂肪 | 40-100g |
| 碳水 | 200-400g |
| 钠 | <2000mg |

## 快餐策略

- 选择烤制而非油炸
- 跳过含糖饮料
- 控制份量
- 当天其他餐食均衡

## 输出指导

- 标记正面和负面营养指标
- 提供替代建议
- 不要因为偶尔的放纵而指责用户
