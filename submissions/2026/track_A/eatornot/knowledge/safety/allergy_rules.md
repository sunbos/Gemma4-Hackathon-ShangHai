# Allergy Rules

## Common Food Allergens

### Major Allergens (Big 8)
- Milk/Dairy
- Eggs
- Peanuts
- Tree nuts
- Fish
- Shellfish
- Wheat
- Soy

### Fast Food Risk Items
- Burgers: wheat (bun), dairy (cheese)
- Chicken: wheat (breading)
- Desserts: dairy, eggs, wheat
- Sauces: may contain various allergens

## Allergy Detection

### User Input
- Ask about known allergies during profile setup
- Store allergy list in user profile
- Check against all menu items

### Matching Process
- Compare user allergies to item tags
- Flag any potential conflicts
- Warn user before ordering

## Response Protocol

### If Allergy Detected
1. Flag the item as potential risk
2. Warn user clearly
3. Suggest safe alternatives
4. Never auto-order allergen-containing items

### If No Allergies
- Confirm "No allergy conflicts detected"
- Proceed with normal flow

## Limitations
- We cannot guarantee cross-contamination
- User should verify with restaurant
- When in doubt, avoid the item

## Language
- Use clear, direct warnings
- Don't minimize allergy risks
- Suggest alternatives positively
