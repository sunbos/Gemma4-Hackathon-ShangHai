- Vercel 已注册 `__server` Serverless Function。
- Gemma 4 Cloud adapter 可达，当前模型为 `gemma-4-26b-a4b-it`。
- 本次 Cloud 连通性测试延迟约 3959 ms。
- TypeScript 类型检查通过。
- 生产构建通过。
- 端侧可行性模拟可运行，并明确标注不是实际硬件 benchmark。
- 家长端默认隐藏儿童原话、事件 ID、精确次数和行为侧写。
- 高风险事件绕过天气隐喻，进入明确的家长安全警报。
- 专业支持交接必须由家长授权，并明确当前为演示流程。

## 提交前必须完成

1. **立即轮换 Gemma API Key**
   - 此前调试终端截图曾显示完整 API Key，应视为已经暴露。
   - 在 Google AI Studio 撤销旧 Key 并创建新 Key。
   - 更新本地 `.env.local` 与 Vercel `GEMINI_API_KEY`。
   - 确认公开视频、截图、仓库与聊天记录中不再使用旧 Key。

2. **录制并提交 5 分钟以内的视频**
   - 使用 `docs/DEMO_SCRIPT.md`。
   - 录制前关闭终端中的 API Key、代理地址和个人账户信息。
   - 使用固定演示场景，避免现场输入造成不可控结果。

3. **准备评委可访问的代码仓库**
   - 上传当前项目。
   - 不上传 `.env.local`、API Key 或本机隐私文件。
   - 在仓库首页放置 Demo URL、视频 URL、README 和技术报告链接。

4. **提交前进行一次完整人工回归**
   - 正常分享。
   - 隐性压力与 Text-only Rules 对比。
   - 生气场景使用 `validate_and_contain`。
   - 高风险场景触发家长警报。
   - 家长授权专业支持交接预览。
   - Family Climate 页面隐藏事件级数据。
   - Cloud Climate 超时不影响端侧响应。
